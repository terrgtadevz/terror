### ❌ Game Crashes on Startup with Loader Active

**Cause:** Outdated build or conflicting background software.

- Verify game files through the game launcher
- Delete `C:\TerrorMenu\logs\` folder and restart

### ❌ Config Resets After Every Launch


### ❌ Anti-Virus Flags the Loader

**Cause:** All game modification tools trigger heuristic AV flags due to process injection behavior — this is expected.

- Add `C:\TerrorMenu\` to your AV exclusion list
- This is standard for all injection-based tools, not a sign of malware

### ❌ Overlay Appears Behind Game Window

**Cause:** Fullscreen exclusive mode blocks overlay rendering.

**Fix:** Switch GTA V to **Borderless Windowed** mode in graphics settings.
